**Aufnahme-Tipps**
- Fenster auf ca. **1440 × 900** (oder größer) ziehen, damit Zwei-Spalten-Layouts (Korrektur, Hilfe) ihre volle Breite zeigen.
- **Dunkles ODER helles Theme** durchgehend gleich wählen (einheitlicher Look).
- Mit **Testdaten** (Klasse TEST-7a) oder `samples/`-Abgaben füllen, damit Diagramme/Listen nicht leer sind.
- Keine echten Schülernamen im Bild (Testdaten/Pseudonyme verwenden).
- Format **PNG**.
